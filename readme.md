[![Build Status](https://travis-ci.org/Themimitoof/Wolfkit.svg)](https://travis-ci.org/Themimitoof/Wolfkit)

Une page de présentation (possédant +80% des éléments du framework) sont affichés et expliqués. Elle est disponible à cette addresse : [http://themimitoof.github.io/Wolfkit](http://themimitoof.github.io/Wolfkit)

If you are a CSS badass (or simple developer), you can read the code and use it on your projects :)

## How to minify CSS
First, do you need to install gulp

```
npm install -g gulp
```

Next, install dependences
```
npm install
```

Start the script
```
gulp
```

And you will find the minified CSS on ``dist``folder.
